Unfold statement: predicate name, predicate args, perm_amount, variant of enum.

Obtain(Expr, Position)

Obtain: conjunction of Expr::PredicateAccessPredicate or Expr::FieldAccessPredicate They will be used by the fold/unfold algorithm

BeginFrame

Mark a CFG point in which all current permissions are framed out They will be used by the fold/unfold algorithm

EndFrame

Mark a CFG point in which all the permissions of a corresponding BeginFrame are framed in They will be used by the fold/unfold algorithm

TransferPerm(Expr, Expr, bool)

Move permissions from a place to another. This is used to restore permissions in the fold/unfold state when a loan expires.

The last argument indicates if the transfer is unchecked. Unchecked transfer is used for encoding shared borrows which can be dangling and, therefore, we cannot use the safety checks.

PackageMagicWand(Expr, Vec<Stmt, Global>, String, Vec<LocalVar, Global>, Position)

Package a Magic Wand Arguments: the magic wand, the package statement’s body, the label just before the statement, and ghost variables used inside the package statement.

ApplyMagicWand(Expr, Position)

Apply a Magic Wand. Arguments: the magic wand.

ExpireBorrows(DAG)

Expire borrows given in the reborrowing DAG.

If(Expr, Vec<Stmt, Global>, Vec<Stmt, Global>)

An if statement: the guard and the ‘then’ branch.

Downcast(Expr, Field)

Inform the fold-unfold algorithm that at this program point a enum type can be downcasted to one of its variants. This statement is a no-op for Viper. Arguments:

  • place to the enumeration instance
  • field that encodes the variant
