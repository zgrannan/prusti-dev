pub enum SpecificationItem<T> {
    Empty,
    Inherent(T),
    Inherited(T),
    Refined(T, T),
}
Expand description

A specification, such as preconditions or a #[pure] annotation. Contains information about the refinement of these specifications.

Variants

Empty

Represents an empty specification, i.e. when the user has not defined the property

Inherent(T)

Represents a specification typed by the user

Inherited(T)

Represents a specification which was inherited from somewhere (e.g. from a trait method)

Refined(T, T)

Represents an item which was refined This happens for example if the user annotates a trait method and its implementation method with a precondition. This variant then holds the trait’s precondition and the impl’s precondition.
